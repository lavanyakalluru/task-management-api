# Task Management API (Spring Boot)

## Overview
A backend system built using Spring Boot that allows users to manage tasks with full CRUD functionality.

## Features
- Create, update, delete tasks
- Retrieve all tasks
- MySQL database integration
- Clean layered architecture

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL

## API Endpoints
POST /tasks
GET /tasks
GET /tasks/{id}
PUT /tasks/{id}
DELETE /tasks/{id}

## Result
Working REST API tested using Postman with proper database integration.
